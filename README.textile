The Apigee API Console for Chrome
-----------------
The Apigee API Console is the best way to explore, test, and learn an API. This Chrome extension extends that power to the Chrome Omnibar context.

Usage
--------
Just type apigee + tab in the omnibar:
!http://github.com/apigee/API-Test-Console/raw/master/doc-images/step1.jpg!


Start typing the API url you want to test:
!http://github.com/apigee/API-Test-Console/raw/master/doc-images/step2.jpg!


Press enter run the API request in the API Console:
!http://github.com/apigee/API-Test-Console/raw/master/doc-images/step3.jpg!

Additionally, you can use the context menu to launch any url or text selection in the API Console:
!http://github.com/apigee/API-Test-Console/raw/master/doc-images/apigee-context-menu.jpg!

This extension suggests API urls for Twitter, Facebook, and SimpleGeo APIs, with more coming soon!

Click on the Apigee 'a' icon to make all endpoint links on a page become testable links into the Apigee API Console:
!http://github.com/apigee/API-Test-Console/raw/master/doc-images/apigify_links.png!

Installation
------------
Make sure you are running Chrome version 7 or higher and that [experimental extensions APIs are enabled](http://code.google.com/chrome/extensions/dev/experimental.html#using)

Requirements
------------
[Google Chrome](http://www.google.com/chrome) or Chromium version 7+.

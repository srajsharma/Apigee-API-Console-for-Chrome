apigee anywhere
-----------------
apigee provides a great tool to explore apis, this is a small extension to use its power in the chrome omnibar context.

Usage
--------
Just type test + tab in the omnibar
!http://github.com/apigee/API-Test-Console/raw/master/doc-images/step1.jpg!


Type the api url you want to test
!http://github.com/apigee/API-Test-Console/raw/master/doc-images/step2.jpg!


Press enter to see apigee console with the api url
!http://github.com/apigee/API-Test-Console/raw/master/doc-images/step3.jpg!

This extension adds a context menu for all the urls and text selection
!http://github.com/apigee/API-Test-Console/raw/master/doc-images/apigee-context-menu.jpg!

This extension suggests api urls for twitter, facebook apis as of now

Click on the apigee icon to apigify text links in any api docs pages, observe the text links become apigee console links
!http://github.com/apigee/API-Test-Console/raw/master/doc-images/apigify_links.png!

Installation
------------
Make sure you are running Chrome version 7 or higher and that [experimental extensions APIs are enabled](http://code.google.com/chrome/extensions/dev/experimental.html#using)

Requirements
------------
[Google Chrome](http://www.google.com/chrome) or Chromium version 7+.
